# Security Policy

## Reporting Vulnerabilities

If you discover a security issue in these plugins, please report it via GitHub Issues or email.

## Verification

To verify you have the official plugins:
- Repository: `asochan8663/claude-plugins`
- Marketplace name: `asochan-ai`
- All other repositories claiming to be "asochan-ai" plugins are unauthorized.

## Plugin Safety

These plugins contain SKILL.md files (Markdown instructions for AI agents). They do NOT contain:
- Executable code
- API keys or tokens
- Network access (all operations are local)

Review any SKILL.md content before installation.
